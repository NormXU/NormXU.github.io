sizeof 对类型名使用时候，应将名称放在括弧内，对变量使用时，则无需放在括弧内

sizeof(int) >> "int is 4 bytes"
short n_short = SHRT_MAX;
sizeof n_short >> 2 bytes

cout.put()成员函数 

char ch;
cout.put(ch); // using cout.put() to display a char constant
最早的时候，我们把字符常量存储为int型，4 bytes = 16位，而char是8位，因此，在执行char ch = 'M'的时候，左边是8位，右边是16位，需要复制'M'的8位给到ch中，因此，当时需要使用cout.put()来将'M'中的8位打印成char型字符。后来修复了这个bug，把字符常量存储为char型，所以这个已经不常用了。
