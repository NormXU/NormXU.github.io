cout.put()成员函数 

char ch;
cout.put(ch); // using cout.put() to display a char constant
最早的时候，我们把字符常量存储为int型，4 bytes = 16位，而char是8位，因此，在执行char ch = 'M'的时候，左边是8位，右边是16位，需要复制'M'的8位给到ch中，因此，当时需要使用cout.put()来将'M'中的8位打印成char型字符。后来修复了这个bug，把字符常量存储为char型，所以这个已经不常用了。

指针
指针存储的值是值的地址，而非值的本身。对于一个变量，在它的前面加上&就可以获悉它的地址。 *运算符被称为间接值或者接触引用运算符，被用于指针。指针的关键作用在于：在运行阶段分配未命名的内存以存储值。 

int* p_update;  // Declare a pointer to int type
int update = 6;
p_update = & update;

int* pt = & update; //也可以这样初始化值
程序告诉new，需要为哪种数据类型分配内存，并使用指针来访问这个值 

int* pt = new int;
*pt = 1001;

//使用完毕后，能够将其归还给内存池
delete pt

// new还可以用于建立动态数组
int* psome = new int [10];
delete [] psome
程序告诉new，需要为哪种数据类型分配内存，并使用指针来访问这个值 

double* p3 = new double [3];
p3[0] = 0.5;
p3[1] = 0.1;
p3[2] = 0.7;

p3 = p3 + 1;
cout << p3[0] << std::endl; // 输出是0.1
p3 = p3 - 1;
delete [] p3;
return 0
相邻int型指针相差2~4个字节，而对pointer做加法后； p3 = p3 + 1，它将指向下一个元素 p3可以作为整个数组名来使用，且是指向第一个元素的地址值。因此，指针变量加1后，增加的量等于他指向的类型的字节数。

对于没有名称，只知道地址的结构，若想去调用它的成员，则需要'->' 或如上述代码所示 

const ana_year* arp[3] = {&s01, &s02, &s03};
// arp是一个数组，这里面数组的所有元素都是只想ana_year的指针
arp则是指向指针的指针， &arp[0] = arp = const ana_year*

//我们可以创造指针的指针，也就是指向const ana_year*的指针
const ana_year** ppa = arp
// 但这么写极其复杂，因此我们可以使用auto关键字来让编译器自己判断指针的类型
auto ppa = arp;

cout << (*(ppa+1))->year << std::endl;
