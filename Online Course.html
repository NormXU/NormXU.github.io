yield python的使用

#生成斐波那契数列
def fab(max): 
    n, a, b = 0, 0, 1 
    while n < max: 
        yield b 
        # print b 
        a, b = b, a + b 
        n = n + 1 
		
#在for循环中调用
for n in fab(5): 
    print n 

#也可以使用next()方法查看下进程
>>> f = fab(5) 
>>> f.next() 
1 
>>> f.next() 
1 
>>> f.next() 
2 
>>> f.next() 
3 
>>> f.next() 
5


python 解释器会将其视为一个 generator，调用 fab(5) 不会执行 fab 函数，而是返回一个 iterable 对象！在 for 循环执行时，每次循环都会执行 fab 函数内部的代码，执行到 yield b 时，fab 函数就返回一个迭代值，下次迭代时，代码从 yield b 的下一条语句继续执行， 而函数的本地变量看起来和上次中断执行前是完全一样的，于是函数继续执行，直到再次遇到 yield。

K-means clustering
Finding K centroid. SO, basically, what group a given data point belongs to is defined by which centroid it is closest to.
  • - Randomly pick K centroids
  • - Assign each data point to the centroid it is closest to
  • - Recompute the centroids based on the average position of each centroid points
  • - iterate until points stop changing assignment to centroid

Gini不纯度
Gini不纯度的值是：1-每个标签占总数的比例的平方和.
总的集合D被分为两个集合D1，D2，假设见面为1，不见面为0

那么D1的不纯度为1-f1^2-f0^2，总数为5，见面的占了全部，则f1=1，f0=0，结果为0

D2的不纯度为1-f1^2-f0^2，f1=0.8，f0=0.2，结果为0.32

ok，那么整个分类结果的Gini不纯度就是D1/D与0的乘积 加上 D2/D与0.32的乘积，为0.16
Gini值代表了某一个分类结果的“纯度”，我们希望结果的纯度很高，这样就不需要对这一结果进行处理了。 从以上分析可以看出，Gini值越小，纯度越高，结果越好。

Q-Learning
- A set of environment states s
- A set of possible actions in those states a
- A value of each state/action Q
As bad things happen after a given state/action, reduce Q
As rewards happen after a given state/action, increase Q

Exploration scheme
- Always chossing the action for a given state with highest Q

Better Way:introducing an epsilon term
If a random number is less than epi

MDPs(Markov Decision Process): describe our exploration algorithm
State: s, s'(next state that we encounter)
Q: Reward Function
Ra(s, s'): moving from state s to another s' get what reward associated with it

Bias and Variance
Bias: how far removed the mean of your predicted values is from the real answer
Variance:how scattered your predicted values are from the real answer
左边:high bias, low variance; 右边:low bias, high variance
Increase K in KNN decrease variance and increase bias
A single decision is prone to overfitting, high variance; bur random forest decresea that variance

def reject_outliers(data):
    u = np.median(data)
    s = np.std(data)
    filtered = [e for e in data if (u - 2 * s < e < u + 2 * s)]
	# 对于每一个在data中的数据，当满足if条件的时候，插入filtered
    return filtered
